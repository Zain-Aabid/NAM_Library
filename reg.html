MobiCMS v1.2.0

This is MobiCMS Dynamic page loader block. There three option to choose on the Page Name Access Type and they are explained below:


EXTERNAL PAGE NAME: if you want to create a Dynamic url page that you can control true the url query variable called "pagename"
INTERNAL PAGE NAME : if you just want to create a Static url page but dynamic page content
ADVANCE PHP EMBED PAGE: if you want to create an embedded Dynamic url page that you can control true the url query variable called "pagename", with dynamic page title and description.(This option required that you have to change some static page value to Mobims dynamic content). And to have a dynamic page setup, following below instructions strictly.

1. Change your page extension from .html to .php
2. paste this code  <?php  include "mobicms/before_html_tag.php";?> on the page Before  <!DOCTYPE>, <html> and <head> tag
3. paste this code  <?php  echo $title;?> on the Page Title tag
4. paste this code  <?php  echo $description;?> on the Page Description tag
5. Finally Some 3rd party code editor escape <?php to <!--?php and ?> to ?--> as Mobicms is built upon PHP therefore making it impossible for Mobicms to load your dynamic page content from the database.
Solution is: After publishing your website and you noticed that your backend page is not showing up, look on the page source code, with any editor(Notepad ++ Preferably)  for escaped code which should be on same line <!--?php should be change to <?php and ?--> to ?> and the issue will be fixed

Note: You will have to disable this notice through the settings Cog before publishing your website to remove this information from your page
Also note: that  you can access Mobicms Backend after publish via www.yourdomain.com/mobicms and also you will need a php supported server  version 5.4 and above to use mobicms backend extension.
